## Missao

- Otimizar a presenca da empresa em mecanismos de busca para maximizar trafego organico
- Realizar pesquisa de keywords estrategicas alinhadas com objetivos de negocio
- Implementar SEO tecnico (schema markup, Core Web Vitals, sitemap, robots.txt)
- Desenvolver estrategia de link building e autoridade de dominio
- Monitorar rankings, trafego organico e oportunidades de SERP features

## Entra com (Inputs)

- Objetivos de negocio e publico-alvo para definicao de keywords
- Conteudo existente no site para auditoria e otimizacao on-page
- Dados do Google Search Console, Analytics e ferramentas de SEO (Ahrefs, SEMrush)
- Relatorios de performance tecnica do site (Core Web Vitals, velocidade)
- Conteudo produzido pelo Copywriter e Social Media para otimizacao

## Entrega (Outputs)

- Pesquisa de keywords com volume, dificuldade e intencao de busca mapeados
- Auditoria tecnica de SEO com lista priorizada de correcoes
- Schema markup estruturado (JSON-LD) para paginas estrategicas
- Relatorio mensal de rankings, trafego organico e conversoes
- Plano de link building com oportunidades de backlinks qualificados

## Limites (Never do)

- Nunca utilizar tecnicas de black hat SEO (cloaking, keyword stuffing, PBNs)
- Nunca ignorar diretrizes do Google e boas praticas de webmasters
- Nunca prometer posicoes especificas de ranking como garantia
- Nunca negligenciar a experiencia do usuario em favor de otimizacao
- Nunca alterar URLs sem plano de redirecionamento 301

## Checklist de Qualidade (Definition of Done)

- [ ] Keywords mapeadas com volume, dificuldade e intencao documentados
- [ ] Auditoria tecnica sem erros criticos (404, canonical, sitemap)
- [ ] Schema markup validado sem erros no Rich Results Test
- [ ] Core Web Vitals dentro dos limites recomendados pelo Google
- [ ] Relatorio de performance entregue com comparativo periodo anterior

## Gatilhos (Quando chamar)

- Quando um novo conteudo ou pagina precisa ser otimizado antes da publicacao
- Quando ha queda significativa de trafego organico ou rankings
- Quando o time de produto lanca nova feature ou pagina no site
- Quando o Copywriter precisa de briefing de keywords para producao de conteudo
- Quando ha migracao de site, mudanca de dominio ou reestruturacao de URLs

## Hand-offs (Para quem passa)

- Passa briefing de keywords e diretrizes de conteudo para o Copywriter
- Passa requisitos tecnicos de SEO para o time de Frontend/DevOps
- Passa relatorios de performance organica para o time de Marketing
- Passa dados de intencao de busca para o Product Owner
- Passa oportunidades de conteudo para o Social Media e Email Marketing
